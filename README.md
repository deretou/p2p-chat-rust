# P2P chat
P2P chat with Rust

## Running Locally

Make sure you have [Rust](https://www.rust-lang.org/tools/install) and [Cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html) installed.

```
git clone https://github.com/deretou/p2p-chat-rust.git
cd p2p-chat-rust
```

Run the server
```
cd server
cargo run
```

Run the client
```
cd client
cargo run
```
